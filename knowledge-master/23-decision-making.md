# Decision Making

How to make better decisions — frameworks, biases, and processes for high-stakes and everyday choices.

## Core Frameworks

**Rational Decision Model:** Define problem → Identify criteria → Weight criteria → Generate alternatives → Evaluate → Select → Implement → Review.

**Cynefin Framework:** Simple (known knowns — best practice), Complicated (known unknowns — expert analysis), Complex (unknown unknowns — probe-sense-respond), Chaotic (unpredictable — act-sense-respond), Disorder (don't know which domain you're in).

**OODA Loop (Boyd):** Observe → Orient → Decide → Act. Speed matters. Continuous cycle.

**First Principles:** Break a problem down to its fundamental truths, then reason up from there. Opposite of reasoning by analogy.

**Inversion:** Instead of asking how to achieve a goal, ask what would guarantee failure — then avoid those things.

## Biases & Heuristics

**Cognitive biases:** Confirmation (seek evidence for beliefs), anchoring (first info dominates), availability (ease of recall = frequency), hindsight ("I knew it all along"), overconfidence (we think we're above average).

**Heuristics:** Mental shortcuts. Useful but error-prone. Recognize when they're appropriate vs when they'll mislead.

## Decision Types

- **Reversible vs irreversible (Bezos):** Reversible: decide fast, undo if wrong. Irreversible: slow down, deliberate more.
- **High stakes / low stakes:** Match decision effort to decision impact.
- **Time pressure:** When speed is critical, accept satisficing over optimizing.

## Personal Framework

1. Do I have enough info? If not, what's missing?
2. What would I tell my best friend in this situation?
3. What would Future Me think of this decision?
4. What's the downside and can I survive it?
5. If I had to decide in 5 minutes, what would I choose?
6. Does this align with my principles?
