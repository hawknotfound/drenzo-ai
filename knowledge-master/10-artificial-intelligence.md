# Artificial Intelligence

Understanding AI systems — how they work, their capabilities, limitations, and trajectory.

## Core Concepts

**Machine Learning:** Systems that improve with data. Supervised (labeled data), unsupervised (patterns), reinforcement (rewards).

**Neural Networks:** Layers of interconnected nodes. Weights adjusted via backpropagation. Universal function approximators.

**Deep Learning:** Multi-layer neural networks. CNNs for vision, RNNs/Transformers for sequence data, GANs for generation.

**LLMs (Large Language Models):** Transformers trained on massive text. Next-token prediction creates emergent abilities: reasoning, translation, coding, etc.

## Key Models & Paradigms

- **GPT (OpenAI):** Decoder-only transformer. InstructGPT, ChatGPT, GPT-4, o1 (reasoning).
- **Claude (Anthropic):** Constitutional AI. Focus on safety and helpfulness.
- **Gemini (Google):** Multimodal (text, image, audio, video).
- **DeepSeek (OpenCode Zen):** Open-weight models. V4 Flash for speed, R1 for reasoning.
- **Open-source:** LLaMA, Mistral, Qwen, Phi.

## Prompt Engineering

- Clear instruction, context, format specification
- Chain-of-thought for reasoning tasks
- Few-shot examples for pattern learning
- Role prompting for persona adoption
- System prompts control behavior at the highest level

## Limitations

- No true understanding — pattern matching
- Hallucinations — confident wrong answers
- Context window limits
- No persistent memory (unless engineered)
- Biases from training data
